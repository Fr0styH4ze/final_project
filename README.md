For my final project I decided to make a simple e-commerce Flutter app that allows users to browse products, view categories, and manage a shopping cart. User authentication and data storage are handled with Supabase.

---

## Features

- User signup and login with Supabase authentication
- A simple splash screen to greet the user
- Browse products by categories 
- View product details
- Add products to shopping cart
- Remove products from shopping cart
- Universal cart button
